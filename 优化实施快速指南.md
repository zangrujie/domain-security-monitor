```python
from concurrent.futures import ThreadPoolExecutor, as_completed

def optimized_batch_query(domains, max_workers=10):
    """优化后的批量查询"""
    results = []
    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        # 提交所有任务
        futures = {executor.submit(query_domain, domain): domain 
                   for domain in domains}
        
        # 收集结果
        for future in as_completed(futures):
            results.append(future.result())
    
    return results
```

### 步骤4：DNS探测优化（第5天）
```python
# 创建 dns_optimizer.py
import asyncio
import aiodns

class DNSOptimizer:
    def __init__(self):
        self.resolver = aiodns.DNSResolver()
        self.cache = {}
    
    async def bulk_resolve(self, domains, concurrency=100):
        """批量DNS解析"""
        semaphore = asyncio.Semaphore(concurrency)
        
        async def resolve_one(domain):
            async with semaphore:
                try:
                    result = await self.resolver.query(domain, 'A')
                    return domain, result
                except Exception as e:
                    return domain, None
        
        tasks = [resolve_one(domain) for domain in domains]
        return await asyncio.gather(*tasks)
```

## 常见问题解决

### Q1：优化后准确率下降？
**解决方案**：
1. 逐步调整参数，避免激进优化
2. 保留原始算法作为备选
3. 实施A/B测试验证效果

### Q2：并发导致资源耗尽？
**解决方案**：
1. 实施资源限制和监控
2. 添加熔断机制
3. 实现优雅降级

### Q3：缓存数据不一致？
**解决方案**：
1. 实现缓存版本控制
2. 添加数据校验机制
3. 定期清理过期缓存
